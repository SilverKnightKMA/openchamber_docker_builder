## Canonical Compare Rules

| Installed vs Pinned | Behavior | Action |

| equal | skip | No-op, skip |

## Checksum Policies

| Policy | Description | Used By |
|--------|-------------|---------|
| requireUpstreamChecksum | Must use authoritative upstream checksum file (checksums.txt, SHA256). GitHub digest NOT acceptable. | gh, yq (if upstream checksum exists), actionlint, hadolint, ruff, scc, clang-format, cmake |
| allowGithubDigest | No upstream checksum exists. GitHub automatic asset digest (June 2025+) is acceptable fallback. | marksman, clangd, protobuf-compiler |
| upstreamOrGithubDigest | Prefer upstream checksum; fall back to GitHub digest if no upstream checksum. | release_binaries family |
| upstreamSha256File | Upstream publishes dedicated SHA256 file. | go_toolchain, cmake |
